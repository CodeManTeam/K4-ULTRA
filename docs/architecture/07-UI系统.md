## 4. 组件生命周期

### 4.1 生命周期方法

```javascript
class MyComponent extends React.Component {
    // 构造函数
    constructor(props) {
        super(props);
        this.state = { ... };
    }
    
    // 组件挂载后
    componentDidMount() {
        // 通常用于：
        // - 注册事件监听
        // - 初始化 Blockly 工作区
        // - 订阅 Redux store
        // - 发起网络请求
        
        // 示例：
        this.workspace = Blockly.inject(this.container, config);
        this.unsubscribe = store.subscribe(this.handleStoreChange);
    }
    
    // 组件更新前
    componentDidUpdate(prevProps, prevState) {
        // 通常用于：
        // - 响应 props 变化
        // - 更新 Blockly 工作区
        // - 刷新数据
        
        // 示例：
        if (prevProps.blocks !== this.props.blocks) {
            this.updateWorkspace(this.props.blocks);
        }
    }
    
    // 组件卸载前
    componentWillUnmount() {
        // 通常用于：
        // - 移除事件监听
        // - 清理定时器
        // - 取消订阅
        // - 销毁 Blockly 工作区
        
        // 示例：
        this.workspace.dispose();
        this.unsubscribe();
    }
    
    // 渲染
    render() {
        return <div>...</div>;
    }
}
```

### 4.2 常见生命周期模式

**初始化 Blockly 工作区**：
```javascript
componentDidMount() {
    // 初始化 Blockly
    this.workspace = Blockly.inject(this.container, {
        toolbox: this.props.toolbox,
        grid: { spacing: 20, length: 3, colour: '#ccc' },
        zoom: { controls: true, wheel: true },
        trashcan: true
    });
    
    // 添加变化监听
    this.workspace.addChangeListener(this.handleWorkspaceChange);
    
    // 加载积木数据
    if (this.props.blocks) {
        Blockly.json.json_to_workspace(this.props.blocks, this.workspace);
    }
}

componentWillUnmount() {
    // 清理
    this.workspace.dispose();
}
```

**订阅 Redux Store**：
```javascript
componentDidMount() {
    // 订阅状态变化
    this.unsubscribe = store.subscribe(() => {
        const state = store.getState();
        this.handleStateChange(state);
    });
}

componentWillUnmount() {
    // 取消订阅
    this.unsubscribe();
}
```

**事件监听**：
```javascript
componentDidMount() {
    // 注册事件
    document.addEventListener('keydown', this.handleKeyDown);
    window.addEventListener('resize', this.handleResize);
}

componentWillUnmount() {
    // 移除事件
    document.removeEventListener('keydown', this.handleKeyDown);
    window.removeEventListener('resize', this.handleResize);
}
```

### 6.2 主题切换

```javascript
// 应用主题
function applyTheme(theme) {
    // 更新 Blockly 主题
    Blockly.mainWorkspace.setTheme(theme);
    
    // 更新 CSS 变量
    document.documentElement.style.setProperty('--workspace-color', theme.colours.workspace);
    document.documentElement.style.setProperty('--toolbox-color', theme.colours.toolbox);
    
    // 刷新渲染
    Blockly.utils.svg_resize(Blockly.mainWorkspace);
}
```

### 7.2 国际化使用

```javascript
// 获取翻译文本
Blockly.Msg.MOTION_MOVESTEPS = "移动 %1 步";

// 切换语言
Blockly.utils.set_language("zh-CN");
Blockly.Msg = Object.assign(Blockly.Msg, languagePack);
```

### 7.3 动态语言切换

```javascript
// 切换语言
function switchLanguage(lang) {
    // 加载语言包
    const languagePack = loadLanguagePack(lang);
    
    // 更新 Blockly 消息
    Blockly.Msg = Object.assign(Blockly.Msg, languagePack);
    
    // 刷新 UI
    refreshUI();
}
```

## 8. 响应式设计

### 8.1 布局自适应

```javascript
// 监听窗口大小变化
window.addEventListener('resize', () => {
    // 更新 Blockly 画布大小
    Blockly.utils.svg_resize(Blockly.mainWorkspace);
    
    // 更新舞台大小
    stage.resize(window.innerWidth, window.innerHeight);
});
```

### 8.2 布局模式

| 模式 | 说明 | 触发条件 |
|------|------|----------|
| **全屏模式** | 隐藏所有面板 | 窗口最大化 |
| **标准模式** | 显示所有面板 | 默认模式 |
| **紧凑模式** | 缩小面板 | 窗口较小 |
| **移动模式** | 抽屉式布局 | 移动设备 |
