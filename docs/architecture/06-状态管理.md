## 1. 状态管理概述

### 1.1 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| **Redux** | 4.x | 状态管理核心 |
| **React-Redux** | 7.x | React 绑定 |
| **Redux-Saga** | 1.x | 副作用管理 |
| **Reselect** | 4.x | 状态选择器（可选） |

### 1.2 数据流

```
用户操作
    ↓
dispatch(Action)
    ↓
Reducer 处理
    ↓
Store 更新
    ↓
组件重新渲染
```

### 1.3 核心概念

| 概念 | 说明 |
|------|------|
| **Store** | 全局状态树，唯一数据源 |
| **Action** | 状态变更的描述对象 |
| **Reducer** | 纯函数，根据 Action 返回新状态 |
| **Saga** | 处理异步操作的 Generator 函数 |
| **Dispatch** | 发送 Action 触发状态变更 |
| **Subscribe** | 订阅状态变化 |

### 3.5 Reducer 定义示例

```javascript
// theatre_reducer 示例
function theatreReducer(state = initialState, action) {
    switch (action.type) {
        case 'SET_THEATRE_STATE':
            return {
                ...state,
                ...action.payload
            };
        
        case 'ACTION_ADD_BCM_ENTITY':
            return {
                ...state,
                actors: {
                    ...state.actors,
                    [action.payload.id]: action.payload
                }
            };
        
        case 'ACTION_SELECT_ENTITY':
            return {
                ...state,
                current_entity: action.payload
            };
        
        case 'ACTION_ADD_THEATRE_SCENE_STATE':
            return {
                ...state,
                scenes: {
                    ...state.scenes,
                    [action.payload.id]: action.payload
                },
                scenes_order: [...state.scenes_order, action.payload.id]
            };
        
        default:
            return state;
    }
}
```

## 5. Saga 副作用管理

### 5.1 Saga 概述

Redux-Saga 用于处理异步操作：

- 网络请求
- 文件操作
- 定时任务
- 复杂业务逻辑

### 5.2 Saga 示例

```javascript
// 保存项目 Saga
function* saveProjectSaga() {
    while (true) {
        // 等待保存请求
        const action = yield take('action_send_save_request');
        
        // 设置保存状态
        yield put({ type: 'set_save_state', payload: { is_saving: true } });
        
        try {
            // 收集项目数据
            const projectData = yield call(collectProjectData);
            
            // 保存到服务器
            const result = yield call(saveToServer, projectData);
            
            // 保存成功
            yield put({ type: 'set_save_state', payload: { is_saving: false } });
            yield put({ type: 'save_success' });
        } catch (error) {
            // 保存失败
            yield put({ type: 'set_save_state', payload: { is_saving: false, error } });
            yield put({ type: 'save_error', payload: error });
        }
    }
}

// 根 Saga
function* rootSaga() {
    yield all([
        fork(saveProjectSaga),
        fork(loadProjectSaga),
        fork(autoSaveSaga),
        fork(collaborationSaga)
    ]);
}
```

### 5.3 常用 Saga Effects

| Effect | 说明 | 示例 |
|--------|------|------|
| `take` | 等待 Action | `yield take('ACTION_TYPE')` |
| `put` | dispatch Action | `yield put({ type: 'ACTION' })` |
| `call` | 调用函数 | `yield call(fn, arg1, arg2)` |
| `fork` | 非阻塞调用 | `yield fork(saga)` |
| `select` | 获取状态 | `yield select(state => state.xxx)` |
| `delay` | 延迟 | `yield delay(1000)` |
| `takeEvery` | 每次都执行 | `yield takeEvery('ACTION', saga)` |
| `takeLatest` | 只执行最新 | `yield takeLatest('ACTION', saga)` |
| `race` | 竞争 | `yield race({ task, cancel })` |
| `cancel` | 取消任务 | `yield cancel(task)` |

## 6. 状态订阅与更新

### 6.1 React 组件订阅

```javascript
import { connect } from 'react-redux';

// mapStateToProps: 将状态映射到 props
const mapStateToProps = (state) => ({
    currentEntity: state.theatre_state.current_entity,
    actors: state.theatre_state.actors,
    isPlaying: state.runtime_state.is_playing
});

// mapDispatchToProps: 将 dispatch 映射到 props
const mapDispatchToProps = (dispatch) => ({
    selectEntity: (id) => dispatch({ type: 'action_select_entity', payload: id }),
    addActor: (data) => dispatch({ type: 'action_add_bcm_entity', payload: data })
});

// 连接组件
export default connect(mapStateToProps, mapDispatchToProps)(MyComponent);
```

### 6.2 直接订阅

```javascript
// 获取 Store
const store = getStore();

// 订阅状态变化
const unsubscribe = store.subscribe(() => {
    const state = store.getState();
    console.log('状态变化:', state);
});

// 取消订阅
unsubscribe();
```

### 6.3 状态选择器

```javascript
import { createSelector } from 'reselect';

// 基础选择器
const getActors = (state) => state.theatre_state.actors;
const getCurrentEntity = (state) => state.theatre_state.current_entity;

// 派生选择器
const getCurrentActor = createSelector(
    [getActors, getCurrentEntity],
    (actors, currentEntity) => actors[currentEntity]
);
```

## 7. 状态持久化

### 7.1 持久化策略

| 状态 | 持久化方式 | 说明 |
|------|-----------|------|
| 项目数据 | 文件保存 | .bcm / .bcm4 文件 |
| 用户设置 | localStorage | 浏览器本地存储 |
| 临时状态 | 不持久化 | 会话结束后丢失 |

### 7.2 localStorage 使用

```javascript
// 保存设置
localStorage.setItem('editor_settings', JSON.stringify({
    theme: 'dark',
    language: 'zh-CN',
    autoSave: true
}));

// 读取设置
const settings = JSON.parse(localStorage.getItem('editor_settings') || '{}');
```

### 7.3 自动保存

```javascript
// 自动保存 Saga
function* autoSaveSaga() {
    while (true) {
        // 等待 30 秒
        yield delay(30000);
        
        // 检查是否需要保存
        const isModified = yield select(state => state.project_state.is_modified);
        
        if (isModified) {
            // 触发保存
            yield put({ type: 'action_send_save_request' });
        }
    }
}
```

## 附录：常见问题

### Q1: 状态更新但组件不重新渲染

**原因**：`mapStateToProps` 返回了相同的引用

**解决**：
```javascript
// 错误：返回相同引用
const mapStateToProps = (state) => state.theatre_state;

// 正确：返回具体属性
const mapStateToProps = (state) => ({
    currentEntity: state.theatre_state.current_entity
});
```

### Q2: Action 不触发 Reducer

**原因**：Action 类型拼写错误

**解决**：
```javascript
// 使用常量定义 Action 类型
const ACTION_SELECT_ENTITY = 'action_select_entity';
```

### Q3: Saga 不执行

**原因**：Saga 未正确启动

**解决**：
```javascript
// 确保 Saga 已启动
sagaMiddleware.run(rootSaga);
```
